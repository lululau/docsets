multi parameter attributes

If you want to use multi-paramater attributes with Rails, you will need to include an extra Mongoid module to support it. 

class Person
  include Mongoid::Document
  include Mongoid::MultiParameterAttributes
end

An example of this is the date_select or datetime_select helper methods: 

= f.date_select(:birthday, :prompt => {:day => 'Day', :month => 'Month', :year => 'Year'})
will send the date as multi param: birthday(1i) birthday(2i) birthday(3i)
Mongoid, 2009-2011, written and maintained by Durran Jordan.
